# SoloTuner — Privacy policy

Publicly accessible privacy policy for the **SoloTuner** mobile app (Google Play and optional direct/beta installs), published by **PaguSoft**.

- **Repository:** [github.com/PaguSoft/solotuner-privacy-policy](https://github.com/PaguSoft/solotuner-privacy-policy)
- **Policy document:** [index.html](index.html)

**GitHub Pages:** enable Pages on the **`main`** branch, folder **`/` (root)**. The public policy URL will be:

**https://pagusoft.github.io/solotuner-privacy-policy/**

Use that HTTPS URL in Google Play Console and in the app when you link the full privacy policy.

A `.nojekyll` file is included so GitHub serves the HTML as static files without Jekyll processing.

Support: use the contact email in the policy or the in-app Information screen, not GitHub Issues here (unless you want otherwise).
